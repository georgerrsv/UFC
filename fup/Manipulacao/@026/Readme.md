## @026 - Positivo, Nulo ou Negativo?

![](https://raw.githubusercontent.com/qxcodefup/arcade/master/base/026/cover.jpg)

### Entrada
- Leia um valor inteiro do usuário.

### Saída
- Imprima "positivo", "nulo" ou "negativo" de acordo com o valor lido.

### Exemplos

```
>>>>>>>>
5
========
positivo
<<<<<<<<

>>>>>>>>
-1
========
negativo
<<<<<<<<

>>>>>>>>
0
========
nulo
<<<<<<<<
```
