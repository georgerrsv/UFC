## @017 - Maior ovo

![](https://raw.githubusercontent.com/qxcodefup/arcade/master/base/017/cover.jpg)

## Ação

Leia 4 números e imprima o maior valor.

### Entrada:

* Leia quatro valores inteiros do usuário.

### Saída:

* Imprima o maior valor lido.

## Exemplos

```
>>>>>>>>
3
3
2
2
========
3
<<<<<<<<

>>>>>>>>
5
1
8
3
========
8
<<<<<<<<
```
